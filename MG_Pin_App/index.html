MG Pin App

About the App

MG Pin App is a private automation tool used to create Pinterest Pins for my own Etsy product listings. The app helps me create image Pins on my own Pinterest boards and link each Pin to the corresponding Etsy listing.

This app is intended for personal and business use by the account owner only. It does not provide a public service, does not allow other users to sign up, and does not post content on behalf of other Pinterest users.

The app uses Pinterest API access only after authorization through Pinterest OAuth. It may access my Pinterest boards and create Pins on my Pinterest account based on product information and images that I provide locally.

Privacy Policy: https://this-site/MG_Pin_App/privacy-policy/

App Purpose

The purpose of MG Pin App is to help publish organic Pinterest Pins for my own Etsy listings. Each Pin contains an image from my own product mockups and links to the related Etsy listing.

The app does not collect data from Pinterest users, does not sell user data, and does not manage advertising campaigns.
